Installation sur cellulaire

IMPORTANT :
- Pour Android/Chrome, l'installation PWA fonctionne correctement si le dossier est hébergé sur un site HTTPS.
- Si vous ouvrez seulement le fichier HTML téléchargé sur le téléphone, l'icône peut ne pas s'afficher comme vraie application.
- Sur iPhone, utilisez Safari puis Partager > Ajouter à l'écran d'accueil.

Fichiers :
- index.html : application
- manifest.json : infos de l'application
- sw.js : installation/cache PWA
- icon-180.png, icon-192.png, icon-512.png : icônes
